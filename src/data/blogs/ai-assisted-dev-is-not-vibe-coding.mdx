Welcome to me first blog post!

The whole reason you're even reading this is because I decided to dip my toes into using AI to create this very site. The main purpose to understand the following points:

* Is it really that "good"?
* Can it replace software engineers
* Does it really make things quicker
* What are the potential pitfalls

I'd being chatting to Gemini about the potential of doing this project and got a rough idea of what I wanted to achieve, but the actual AI I used for this was Anthropic's current darling Claude, specifically the model Sonnet 4.6, I'd been warned that Opus is very token hungry, and Sonnet is good enough. So I get a Claude Pro subscription for £18 per month and dived in.

So initially I didn't do anything with Claude to actually build something, I main played around with get Claude COde agents setup in docker containers with MCP servers in seperate conatiners, mainly to uderstand how you could automate a SDLC (Software Development Lifecycle) with AI agents (that will probably be a whole different blog post sometime in the future). But then once I was satisifed I knew enough, it was time to build something.
